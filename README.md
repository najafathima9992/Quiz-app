# AWS Serverless Quiz Application

This project is a cloud-based quiz application built using HTML, CSS, JavaScript, AWS EC2, API Gateway, AWS Lambda, and DynamoDB.

## Features
- User-friendly quiz interface
- Score calculation
- API integration using API Gateway
- Result storage in DynamoDB
- Hosted on AWS EC2

## Technologies Used
- HTML
- CSS
- JavaScript
- AWS EC2
- AWS API Gateway
- AWS Lambda
- AWS DynamoDB
